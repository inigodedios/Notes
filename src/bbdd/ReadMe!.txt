SQLite:
- Ventajas
    - Ocupa poca memoria
    - Muy eficiente y rapido
    - Multiplataforma
    - Sin administración / configuración

- Desventajas
    - No admite clausulas anidadas (where)
    - No acceso simultáneo a la vez, ni usuarios
    - Falta de foreign key cuando se crea en modo consola

Pasos a seguir para conectar con bbdd
1. Abrir - crear conexión
2. Crear puntero
3. Ejecutar consulta SQL
4. Manejar los resultados de la consulta (CRUD = Create, Read, Update, Delete)
    - Insertar
    - Leer
    - Actualizar
    - Borrar
5. Verificar cambios
6. Cerrar puntero
7. Cerrar conexión
